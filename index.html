Clase Planeta

Descripción

Modelado de planetas del sistema solar con propiedades físicas y orbitales utilizando principios de POO.

Características Principales

  • Clase TipoPlaneta auxiliar con enumeraciones
  • Cálculo preciso de densidad (masa/volumen)
  • Determinación de planeta exterior basado en UA
  • Validación robusta de entradas por consola
  • Manejo de unidades astronómicas (UA)
  • Interfaz de usuario intuitiva

Atributos Clave

nombre, cantidad_satelites, masa, volumen, diametro, distancia_media_sol, tipo_planeta

Clase Automóvil

Descripción

Representación completa de vehículos con todas sus características y comportamientos mediante POO.

Características Principales

  • Enumerado Color con 8 opciones predefinidas
  • Clases auxiliares TipoCombustible y TipoAutomovil
  • Cálculo de tiempo estimado de llegada
  • Métodos GET/SET para todos los atributos
  • Validaciones exhaustivas de entrada
  • Control de velocidad actual vs máxima

Atributos Clave

marca, modelo, motor, tipo_combustible, tipo_automovil, numero_puertas, cantidad_asientos, velocidad_maxima, color, velocidad_actual

Clase Cuenta Bancaria

Descripción

Sistema bancario con herencia para cuentas de ahorro y corriente, implementando reglas específicas para cada tipo.

Características Principales

  • Herencia: CuentaBancaria → CuentaAhorros/CuentaCorriente
  • Manejo inteligente de sobregiros
  • Estado activo/inactivo automático
  • Cálculo de intereses mensuales
  • Comisiones por exceso de transacciones
  • Validación de operaciones por estado

Funcionalidades Específicas

CuentaAhorros: Activación automática con saldo ≥ $10,000, comisión por retiros excesivos (>4/mes)

CuentaCorriente: Sobregiro permitido, comisión mensual fija, priorización de pago de sobregiro
